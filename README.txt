Задача: Написать программу, которая из имеющегося массива строк формирует новый массив из строк, 
длина которых меньше, либо равна 3 символам. 
Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма. 
Решение:
1. Для создания массива пользователю предлагается ввести элементы  с клавиатуры с разделителем ",".
2. Строка преобразовывается в массив, с помощью метода разделяющего элементы по ",".
3. С помощью цикла, длина каждого элемента массива сравнивается с условием меньше или равна 3 символам.
4. Из элементов имеющих длину от 1 до 3х символов составляется новый массив.
5. Вывод нового массива из строк, длина которых меньше, либо равна 3 символам.
